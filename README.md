# Flowdesk — لوحة مهام الفريق

تطبيق إدارة مهام بواجهة عربية كاملة (RTL)، يعمل بدون باك‑إند (HTML/CSS/JS فقط).

## المزايا

- 👑 **ثلاث أدوار** بصلاحيات مختلفة (مدير / مشرف / موظف)
- 🖥️ **لوحة تحكم كاملة** للمديرين والمشرفين (شبكة موظفين + سحب وإفلات)
- 📱 **تطبيق جوال مخصص** للموظفين (إطار هاتف على سطح المكتب)
- 🔒 **نظام صلاحيات** يفلتر المهام والموظفين حسب الدور والقسم
- 📊 تقارير، تقويم، XP، مستويات، شارات إنجاز
- 💾 يحفظ كل شي محلياً في `localStorage`

## حسابات تجريبية

| الدور | رقم المستخدم | كلمة المرور |
|---|---|---|
| 👑 مدير | `151215` | `1111` |
| 🛡️ مشرف | `152525` | `2525` |
| 👤 موظف | `153030` | `3030` |

## هيكل الملفات

```
├── login.html / login.js            # تسجيل الدخول
├── index.html / app.js              # لوحة المهام (مدير/مشرف)
├── team.html / team.js              # صفحة الفريق
├── calendar.html / calendar.js      # التقويم
├── reports.html / reports.js        # التقارير
├── favorites.html / favorites.js    # المفضلة
├── notifications.html / *.js        # التنبيهات
├── employee.html / employee.js      # ملف موظف
├── my-tasks.html / my-tasks.js      # تطبيق الموظف (جوال)
├── data.js                          # بيانات + صلاحيات + helpers
├── layout.js                        # الـtopbar + auth guard
├── task-modal.js                    # نافذة إنشاء مهمة
├── task-detail-modal.js             # نافذة تفاصيل مهمة
└── styles.css                       # كل الستايل
```

## التشغيل المحلي

```bash
python3 -m http.server 8765
# ثم افتح http://localhost:8765/login.html
```

## النشر على Vercel

```bash
npm i -g vercel
vercel
```

اتبع التعليمات. لا يحتاج بناء — موقع ثابت.

## ملاحظة أمنية

كلمات المرور وأرقام المستخدمين موجودة في `data.js` (عميل فقط). للإنتاج الحقيقي يجب نقل المصادقة لخدمة باك‑إند (Auth0 / Firebase / Supabase…) أو استبدال هذي البيانات بطلب إلى API.
